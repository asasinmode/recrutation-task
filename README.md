# recrutation task

This is a simple layout implemented in Vue. It was made as a job recrutation task.

### [preview on github pages](https://asasinmode.github.io/recrutation-task/)

# development

```bash
# install dependencies
pnpm i

# start dev server
pnpm dev

# build app
pnpm run build

# type check
pnpm run typecheck

# lint
pnpm run lint
```
